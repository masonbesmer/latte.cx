## What Was Built

### Synthwave Route (`/synthwave`)
A new `/synthwave` route added to the TanStack Router app, implementing the
Synthwave Outrun Grid Dashboard as described in the PDF design document.

**Route files:**
- `src/routes/synthwave/route.tsx` — Layout wrapper; mounts SynthwaveBackground, imports CSS
- `src/routes/synthwave/index.tsx` — Main dashboard page with all live-updating panels

**Components:**
- `src/components/synthwave/SynthwaveBackground.tsx` — Fixed Three.js canvas:
  - Scrolling magenta perspective grid (LineSegments, animated Z offset, wrapping)
  - Dark mountain silhouettes at horizon using THREE.ShapeGeometry
  - Ambient pink/cyan particle drift (600 points, AdditiveBlending)
  - UnrealBloomPass (strength 0.8, radius 0.5, threshold 0.05) for strong neon glow
  - CSS gradient sky (deep purple → violet → pink → orange at horizon)
  - CSS sun with chrome gradient + horizontal blind lines
  - Canvas alpha transparent so CSS sky shows through
- `src/components/synthwave/StatCard.tsx` — Per-service stat card:
  - Category-colored neon border (infrastructure=cyan, media=pink, gaming=yellow, etc.)
  - Status orb with pulse animation (healthy/warning/down/starting/idle)
  - CPU and RAM progress bars with color shift (cyan → orange → red)
  - Click to open ServiceModal
- `src/components/synthwave/CPUBars.tsx` — Audio-equalizer style CPU core visualizer
  - 16 animated bars, color shifts per value
  - Animated live data (800ms intervals)
- `src/components/synthwave/NetworkGraph.tsx` — Canvas-based neon network graph
  - Download (cyan) and upload (pink) lines with gradient fill
  - Auto-scales to max value; labeled live readings
  - Updates every 2s with simulated data
- `src/components/synthwave/CircularGauge.tsx` — SVG circular progress rings
  - Orbitron font for percentage, drop-shadow glow
  - Color shifts cyan → orange → red at high values
- `src/components/synthwave/TerminalLog.tsx` — Scrolling terminal-style log
  - Live entries appended every 3.5s from template pool
  - Color-coded levels: ok=green, info=cyan, warn=orange, error=red
  - aria-live=polite for accessibility
- `src/components/synthwave/ServiceModal.tsx` — Service detail modal (click stat card)
  - Escape key / backdrop click to close
  - Shows all service details, version, uptime, link to dashboard

**Data:**
- `src/lib/synthwave-data.ts` — Mock homelab data:
  - 8 services: Proxmox, UniFi, Home Assistant, Plex, Minecraft, Grafana, Gitea, Pi-hole
  - ServiceStatus, ServiceCategory, CATEGORY_COLOR, STATUS_COLOR maps
  - mockCpuCores(), generateNetworkHistory(), initialLogs, newLogTemplates

**Styles:**
- `src/styles/synthwave.css` — All synthwave design tokens and component CSS:
  - Color tokens: --sw-black, --sw-pink, --sw-cyan, --sw-yellow, etc.
  - CSS gradient sky (background-attachment: fixed)
  - Scanlines overlay (::before repeating-linear-gradient)
  - CRT vignette (::after radial-gradient)
  - sw-sun with horizontal blinds effect (::after repeating-linear-gradient)
  - sw-panel with glass-morphism + corner accents + Firefox fallback
  - All component styles: stat cards, cpu bars, network graph, gauges, terminal, modal
  - Keyframes: sw-pulse, sw-blink, sw-slide-in, sw-slide-up, sw-fade-in

## Architecture Notes
- The synthwave route follows the same layout pattern as /cyberpunk (route.tsx + index.tsx)
- SynthwaveBackground uses alpha:true Canvas so the CSS sky/sun gradient shows through
- Three.js grid scrolls by animating horizontal line Z positions each frame (wraps at H_SPACING)
- All live data is simulated via setInterval in the dashboard component with proper cleanup
- routeTree.gen.ts is auto-regenerated by TanStack Router Vite plugin on build/dev
